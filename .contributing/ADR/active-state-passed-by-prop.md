# Active state should be passed by prop instead of guessed by the component

## Context

Discussion: <https://github.com/swisspost/design-system/discussions/6636>

> [!NOTE]
> This ADR is a draft. The decision and consequences are not yet defined — see the discussion for the latest status.

Components like `<post-tabpanel>` or `<post-mainnavigation>` can contain items with an active state that's based on the currently active URL.

## Decision

_To be defined._ The proposal is that the active state should be passed explicitly by prop instead of being guessed by the component (e.g. from the current URL).
