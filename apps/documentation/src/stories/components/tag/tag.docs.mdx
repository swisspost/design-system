## Neutral tag

Use the neutral tag for general-purpose labeling. You can pair it with any icon to provide additional context.
Browse available icons on the <a href="/?path=/docs/0dcfe3c0-bfc0-4107-b43b-7e9d825b805f--docs">icon search page</a>.

## Signal tags

Signal tags communicate status or meaning through color and icon.
Each variant includes a default signal icon, which can be replaced with a custom icon if needed. Note that the icons should be meaningful and relevant to the context of the tag.
