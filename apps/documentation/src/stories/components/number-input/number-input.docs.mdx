## Examples

### Floating Label

Wrap a pair of `<input>` and `<label>` elements in a `<post-number-input class="form-floating">` to enable floating labels.<br/>
But note that the `<input>` element must come first, so we can ensure the correct styles.

Ensure that `placeholder` attribute is set (even with an empty value) so the label can act as a placeholder when no value is set.

<Canvas of={NumberInputStories.FloatingLabel} />

### Small input

For a more compact version of the number input, add the `.form-control-sm` class to the `input`. This variant only works with non-floating labels.
