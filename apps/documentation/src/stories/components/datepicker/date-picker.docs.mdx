## Examples

### Inline calendar

Inline calendars render directly in the page flow instead of a popup.
They work well in dashboards, forms, and any context that benefits from a persistent calendar view.

<Canvas sourceState="hidden" of={DatePickerStories.Inline} />

### Range input calendar

Range input calendars allow users to select a start and end date. These are ideal for booking systems, event planners, or any scenario where date ranges are relevant.

<Canvas sourceState="hidden" of={DatePickerStories.Range} />

## Disabling dates

The Date Picker provides a `cellConfig` property, which is triggered for each calendar cell when rendering.
It allows you to dynamically modify the state of individual cells, including disabling certain dates or applying custom classes.
