## SCSS Imports

To import all styles from the design-system-styles package:

```css dark
@use '@swisspost/design-system-styles/index.scss';
```

To import only the styles required for this component:

```css dark
@use '@swisspost/design-system-styles/basics.scss';
@use '@swisspost/design-system-styles/components/table.scss';
```

## Usage examples

### Table with buttons

When buttons are placed in a table, be sure to use `alignment: middle`, respectively `<tbody class="align-middle">` to nicely align the content and the buttons.
