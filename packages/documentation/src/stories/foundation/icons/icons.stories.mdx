# Icons

<p className="lead">
  The official Swiss Post Icon Library for the web with over 800 high-quality scalable vector
  graphics.

The icons are designed to work with the Swiss Post Design System components. They are SVGs, so they scale and can be styled with CSS.<br/>
Use the grid below to search the available icons.

More information about the icons can be found under:

- The [Icons Package](/?path=/docs/get-started-icons--page) to use them in an **svg-only** approche.
- The [Icons Component](/?path=/docs/components-icon--default) to use them with our **icon** component.
