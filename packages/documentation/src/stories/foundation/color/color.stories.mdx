# Usage

## CSS

CSS variables are globally available on your page.

<div class="alert alert-info alert-md mb-regular">
  Note:With this approach, the text color will not be automatically accessible. Use the{' '}
  <LinkTo kind="utilities-background--background-containers">background utilities</LinkTo> for this
  feature.
</div>

```css dark
.custom-background {
  background-color: var(--post-dark);
}
```

## Sass

Import the color variables from the `core` file with `@use`. This way, the color variables are scoped and don't mix with your own variables.

<div class="alert alert-info alert-md mb-regular">
  Note: With this approach, the text color will not be automatically accessible. Use the{' '}
  <LinkTo kind="utilities-background--background-containers">background utilities</LinkTo> for this
  feature.
</div>

```css dark
@use '@swisspost/design-system-styles/core' as post;

.custom-background {
  background-color: post.$dark;
}
```
