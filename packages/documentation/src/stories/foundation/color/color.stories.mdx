# Usage

## CSS

CSS variables are globally available on your page. When using CSS variables, the text color will not be automatically accessible. Use the background classes for this feature.

```css dark
.custom-background {
  background-color: var(--post-yellow);
}
```

## Sass

Import the color variables from the Design System Styles package from the `core` sass file with `@use`. This way, the color variables are scoped and don't mix with your own variables.

With this approach, the text color will not be automatically accessible. Use the background classes for this feature.

```css dark
@use '@swisspost/design-system-styles/core' as post;

.custom-background {
  background-color: post.$yellow;
}
```
