# Usage

Import the Design System Styles package and choose from one of the provided style files:

- **basics.css**: Just the basic styling for resets, typography and utility classes
- **index.css**: The complete styling for internet facing applications
- **intranet.css**: The complete styling for internal applications

## Class

Apply any of the background classes to your elements. The text color will be automatically set for best contrast.

```html
<div class="bg-aubergine">This text will be white.</div>
```

## CSS

Use one of the CSS variables in your style sheets. When using CSS variables, the text color will not be automatically accessible. Use the background classes for this feature.

```css
.background {
  background-color: var(--post-yellow);
}
```

## Sass

Import the color variables from the Design System Styles package. If you use `@use` to import the vars, the color variables are scoped and don't mix with your own variables.

When using sass variables, the text color will not be automatically accessible. Use the background classes for this feature.

```scss
@use '@swisspost/design-system-styles/core' as post;

.background {
  background-color: post.$yellow;
}
```
