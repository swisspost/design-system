# Icon Health Report

This report provides an overview of the current state of our icon system, highlighting any issues or inconsistencies that need attention.

<Canvas of={IconStories.Icons} />

## Understanding the Report

- **Total Icons**: The total number of icons in our system.
- **Successful**: Icons that passed all checks.
- **Errors**: Icons that encountered errors during processing.
- **Not Found**: Icons that were expected but not found in the system.

### Issue Categories

1. **Wrong ViewBox**: Icons that don't have the standard viewBox of "0 0 32 32".
2. **No Keywords**: Icons lacking descriptive keywords, which may affect searchability.
3. **No SVG**: Expected icons that are missing their SVG files.
4. **Errored**: Icons that encountered specific errors during processing.

If you notice any discrepancies or have questions about this report, please contact the design system team.
