# @swisspost/design-system-styles

[![npm version](https://badge.fury.io/js/@swisspost%2Fdesign-system-styles.svg)](https://badge.fury.io/js/@swisspost%2Fdesign-system-styles)

The styling package, including theming for [Bootstrap](https://getbootstrap.com/) and [ng-bootstrap](https://ng-bootstrap.github.io/#/home) components.

## Installation

<Source code={`npm install @swisspost/design-system-styles`} language="bash" />

## Usage

Import one of our stylesheets into your project (for example into your `/src/styles.scss` file).

<Source
  code={`@use '@swisspost/design-system-styles/<bundle-name>.<extension>';`}
  language="scss"
/>
