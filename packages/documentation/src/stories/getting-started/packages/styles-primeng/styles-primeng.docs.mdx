# @swisspost/design-system-styles-primeng

[![npm version](https://badge.fury.io/js/@swisspost%2Fdesign-system-styles-primeng.svg)](https://badge.fury.io/js/@swisspost%2Fdesign-system-styles-primeng)

The Design System Styles for PrimeNG to style your PrimeNG datatables.

## Prerequisites

Make sure you have already installed and configured **PrimeNG** in your Angular project.
If not, follow the <a href="https://primeng.org/installation">PrimeNG installation guide</a> first.

## Installation

Install the Swiss Post Design System styles for PrimeNG DataTables by running:

<Source code={`npm install @swisspost/design-system-styles-primeng`} language="bash" />
