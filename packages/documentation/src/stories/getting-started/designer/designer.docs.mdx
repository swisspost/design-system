# Getting Started as a Designer

<p className="lead">Welcome to the Swiss Post Design System world!</p>

This page provides links to essential resources that will help you design components,
follow system guidelines, and ensure consistency across our work.

## Figma Resources

Here are the key Figma files that you will need to create and document your designs:

- [**Foundations & Components**](https://www.figma.com/design/JIT5AdGYqv6bDRpfBPV8XR/Foundations-%26-Components-Next-Level?node-id=26-45&node-type=canvas&t=KgHJQrU7azeYeisY-0) -
Access the core components and design building blocks used throughout the system.

- [**Icons**](https://www.figma.com/design/q57uYXNspU7G1pYEnDGTyM/Icons-Next-Level?node-id=0-1&node-type=canvas&t=gp3LWYP1tnC6I8Lb-0) -
Explore our icon set and learn how to integrate them into your designs.

- [**Layout Examples**](https://www.figma.com/design/1U0tACfFegz2s1ThIr4M06/Layout-Examples-Editorial-Next-Level?node-id=2132-12618&node-type=canvas&t=CruHFQfJ7SFf2tXo-0) -
View layout patterns and examples that demonstrate best practices and consistency across designs.
