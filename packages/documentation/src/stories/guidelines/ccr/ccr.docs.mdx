    Use the `ariaLabelledByElements`: This has limited coverage  (<a href="https://caniuse.com/mdn-api_element_arialabelledbyelements"> ariaLabelledByElements Usage</a>) and works only if the referencing element is in the light DOM and the referenced element in the shadowDom.
  </li>
    <li>Use `aria-label` with the label text directly on the `<input>` element. </li>
</ul>
