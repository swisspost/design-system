## Font

Quickly change how fonts are displayed in specific elements and/or their child nodes.<br/>
To do so, just add the class to your element or use the scss-variable in your code.

### Font Family

<Canvas withSource="open">
  <Story name="Font Family">
    <p className="font-sans-serif">Sans Serif text, ...</p>
  </Story>
</Canvas>

#### Set Font Family in scss

```css dark
@use '@swisspost/design-system-styles/core' as post;

.my-element {
  font-family: post.$font-family-sans-serif;
}
```

#### Available values

<AvailableClassesAndVariables
  variants={SCSS_VARIABLES.fontFamilies}
  cssPrefix="font"
  scssPrefix="font-family"
/>

## Text

Define how text-blocks are displayed.

### Color

<div class="alert alert-warning my-4">
  <p>
    The classe <code>text-auto</code> is being deprecated because it is no longer necessary in
    combination with our <LinkTo href="utilities-background">background utilities</LinkTo>.
  </p>
</div>

Colorize text with color utility classes. If you want to colorize links, you can use the `.link-*` classes instead, which comes with styles for `:hover` and `:focus` states.

<Canvas withSource="open">
  <Story
    name="Color"
    decorators={[
      Story => (
        <div onClick={e => e.preventDefault()}>
          <Story />
        </div>
      ),
    ]}
  >
    <p className="text-success">
      Colored text, ...
      <br />
      <a href="#" className="link-danger">
        Colored link, ...
      </a>
    </p>
  </Story>
</Canvas>

#### Set the Color in scss

```css dark
@use '@swisspost/design-system-styles/core' as post;

.my-element {
  color: post.$success;
}
```

#### Available values

### Alignment

Easily realign text with our text alignment utility classes.

<Canvas withSource="open">
  <Story name="Alignment">
    <p className="text-start">Start aligned text</p>
    <p className="text-center">Center aligned text</p>
    <p className="text-end">End aligned text</p>
  </Story>
</Canvas>

#### Responsive alignment

For `start`, `end`, and `center` alignment, responsive classes are available that use the same viewport breakpoints as the grid system.
The responsive alignment classes have the following format: `text-{breakpoint}-{alighment}`.

For more detailed informations about breakpoints read our <LinkTo kind="foundations-grid">grid docs</LinkTo>.

### Wrapping

Manage whether text should wrap or not with the utility classes `.text-nowrap` and `.text-wrap`.

<Canvas withSource="open">
  <Story name="Wrapping">
    <div className="w-50 bg-info">
      <p className=" text-nowrap">
        This text should NOT wrap, saepe excepturi quas nihil repudiandae eius assumenda voluptatem.
      </p>
    </div>
    <div className="w-50 bg-info text-nowrap">
      <p className="text-wrap">
        This text should wrap, saepe excepturi quas nihil repudiandae eius assumenda voluptatem.
      </p>
    </div>
  </Story>
</Canvas>

### Word Break

Prevent long strings of text from breaking your component's layout by using the utility class `.text-break`.

<Canvas withSource="open">
  <Story name="Word Break">

  </Story>

### Transform

Change the `text-transform` manually with the following utility classes: `.text-lowercase`, `.text-uppercase` and `.text-capitalize`.

<Canvas withSource="open">
  <Story name="Transform">

    <p className="text-uppercase">uppercase text, ...</p>
    <p className="text-capitalize">uppercase text, ...</p>
  </Story>
</Canvas>
