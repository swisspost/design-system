# Lists

<p className="lead">HTML lists allow web developers to group a set of related items.</p>

There are three main types of lists, from which you can choose the one that best suits your needs.
Our utilities give you the possibility to style each of them in different ways.

## Unordered List

The HTML `<ul>` tag defines an unordered (bulleted) list.

<Canvas>
  <Story name="Unordered List">
    <ul>
      <li>This is an unorderd list.</li>
      <li>It appears in its default style.</li>
      <li>
        Therefore it is rendered with a bullet point in front of each list item.
      </li>
      <li>
        Nested list:
        <ul>
          <li>This is a nested list</li>
          <li>It is further indented, depending on the depth of nesting.</li>
          <li>Nested lists have different bullet points than their parents.</li>
        </ul>
      </li>
      <li>This item belongs to the parent list.</li>
    </ul>
  </Story>
</Canvas>

### Unstyled

Remove the default list-style and left margin on list items (immediate children only). This only applies to immediate children list items, meaning you will need to add the class for any nested lists as well.

<Canvas withSource="open">
  <Story name="Unordered unstyled">
    <ul className="list-unstyled">
      <li>This is a unstyled list.</li>
      <li>It appears completley unstyled.</li>
      <li>Structurally, it's still a list.</li>
      <li>
        Nested list:
        <ul>
          <li>They are unaffected by the style of its parent.</li>
          <li>So they will still show a bullet.</li>
          <li>And have an appropriate left indent.</li>
        </ul>
      </li>
      <li>This item belongs to the parent list.</li>
    </ul>
  </Story>
</Canvas>

### Inline

Remove a list’s bullets and apply some light margin with a combination of two classes, .list-inline and .list-inline-item.

<Canvas withSource="open">
  <Story name="Unordered inline">
    <ul className="list-inline">
      <li className="list-inline-item">This is a inline list item.</li>
      <li className="list-inline-item">And another one.</li>
      <li className="list-inline-item">And one more.</li>
    </ul>
  </Story>
</Canvas>

## Ordered List

The HTML `<ol>` tag defines an ordered (numbered) list.<br/>
The utilities shown for the `<ul>` element above, can also be used on a ordered list. Therefore we're not going to show this here again.

<Canvas>
  <Story name="Ordered List">
    <ol>
      <li>This is an ordered list.</li>
      <li>It appears in its default style.</li>
      <li>
        Therefore it should be rendered with sequential numbers at the beginning of each list item.
      </li>
      <li>
        Nested list:
        <ol>
          <li>This is a nested list</li>
          <li>It is further indented, depending on the depth of nesting.</li>
          <li>Nested lists numbers are independant form the numbers of their parents.</li>
        </ol>
      </li>
      <li>This item belongs to the parent list.</li>
    </ol>
  </Story>
</Canvas>

## Description List

The `<dl>` HTML element represents a description list. The element encloses a list of groups of terms and descriptions.

<Canvas>
  <Story name="Description List">
    <dl>
      <dt>This is a description list.</dt>
      <dd>It appears in its default style.</dd>
      <dt>Group title</dt>
      <dd>A description list is perfect for defining terms.</dd>
      <dt>Term</dt>
      <dd>Definition for the term.</dd>
    </dl>
  </Story>
</Canvas>

### Alignment

Align terms and descriptions horizontally by using our grid system’s predefined classes (or semantic mixins). For longer terms, you can optionally add a `.text-truncate` class to truncate the text with an ellipsis.
