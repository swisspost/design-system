## Border

The Border utility provides precise control over an element’s borders, allowing you to add, remove, or adjust borders on specific sides or all around using predefined classes.

- `border`: Applies a border on all sides of an element.

- `border-{side}`: Sets the border on a specific side. Supports `border-top` (top edge), `border-end` (inline end, mapped to right in LTR), `border-bottom` (bottom edge), and `border-start` (inline start, mapped to left in LTR).

- `border-color`: Defines the border color.

- `border-width`: Defines the border width.

- `border-opacity`: Defines the border opacity.

## Border Radius

Add `rounded` to an element to easily apply a border-radius on its corners.

- `rounded`: Applies the default border-radius on all corners.
- `rounded-{side}`: Αpplies the default border-radius on a specific side. Supports `rounded-top` (top edge), `rounded-end` (inline end, mapped to right in LTR), `rounded-bottom` (bottom edge), and `rounded-start` (inline start, mapped to left in LTR).
