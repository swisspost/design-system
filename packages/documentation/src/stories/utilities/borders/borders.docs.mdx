# Borders & Radius

## Border

<div className="lead">
  Use border utilities to quickly style the border and border-radius of an element. Great for
  images, buttons, or any other element.
</div>

Border utilities allow precise control over an element’s borders, enabling you to add, remove, or modify borders on specific sides or all around.

- `border`: Applies a border on all sides of an element.

- `border-{side}`: Sets the border on a specific side. Supports border-top (top edge), border-end (inline end, mapped to right in LTR), border-bottom (bottom edge), and border-start (inline start, mapped to left in LTR).

- `border-color`: Defines the border color.

- `border-width`: Defines the border width.

- `border-opacity`: Defines the border opacity. For the opacity to take effect, a border-color must be applied.

## Border Radius

Add `rounded` to an element to easily round its corners.
Use the scaling classes to adjust the size of rounded corners. Available sizes range from 0 (no rounding) to 5 (maximum rounding).

- `rounded`: Applies a border-radius on all corners.
- `rounded-{side}`: Αpplies a border-radius on a specific side. Supports rounded-top (top edge), rounded-end (inline end, mapped to right in LTR), rounded-bottom (bottom edge), and rounded-start (inline start, mapped to left in LTR).
