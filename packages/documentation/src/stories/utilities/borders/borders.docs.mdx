## Border

<div className="alert alert-info">

### Properties

- `border`: Applies a border on all sides.
- `border-top`: Applies a top border.
- `border-end`: Applies a right border (in LTR).
- `border-bottom`: Applies a bottom border.
- `border-start`: Applies a left border (in LTR).
- `border-color`: Sets the border color.
- `border-width`: Sets the border width.
- `border-opacity`: Sets the border opacity.

## Radius

Add `rounded` to an element to easily round its corners.
Use the scaling classes to adjust the size of rounded corners. Available sizes range from 0 (no rounding) to 5 (maximum rounding).

### Properties

- `rounded`: Applies a border-radius on all corners.
- `rounded-top`: Applies a border-radius to the top corners.
- `rounded-end`: Applies a border-radius to the right corners (in LTR).
- `rounded-bottom`: Applies a border-radius to the bottom corners.
- `rounded-start`: Applies a border-radius to the left corners (in LTR).
