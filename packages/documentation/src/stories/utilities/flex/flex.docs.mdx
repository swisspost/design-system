# Flex

<div className="lead">
  Use the flex utility classes to change the display of elements within a .d-flex parent.
</div>
<br />

Flex utilities work alongside the <code>display: flex</code> css class that can be added to any HTML element using the classes `d-flex` or `d-{breakpoint}-flex`.

## Flex direction

The available utility values for the flex direction are the following:

- row (default)
- row-reverse
- column
- column-reverse
  <br />
  Naming convention for flex direction:
- For all breakpoints (<code>{firstBreakpoint}</code> and up): `flex-{value}`
- Starting from a specific breakpoint ({breakpoints.map(([b, isLast]) => (<span key={b}><code>{b}</code>{isLast ? ', ' : ''}</span>))}): `flex-{breakpoint}-{value}`

### Usage

To set the direction of elements within a flex container, simply add a `flex-{value}` class to the container.

<Canvas of={FlexStories.Default} sourceState="shown" />

<section>
  {forEach(['row', 'row-reverse', 'column', 'column-reverse'], function ({ key, value }) {
    return <FlexDirectionContainer key={key} name={value} />;
  })}
</section>
