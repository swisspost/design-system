## Usage on heading Elements

Use the utility classes (`.h1` - `.h6`), to change the appereance of heading elements.

<Canvas withSource="open">
  <Story name="Heading Elements">
    <h6 className="h3">Heading level 1 with h3 styles...</h6>
  </Story>
</Canvas>

## Usage on other Elements

Use the utility classes, to make an element look like a heading.
