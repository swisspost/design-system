In the example above, the `.section` class ensures that the background-colored wrapper never exceeds 1440px.
Meanwhile, the `.container` class keeps the content aligned consistently with the rest of the page.

## Edge alignment

Inside the `.section`, you can use helper classes to align elements with either the edges of the `.section` or the `.container`.
These helpers make it easy to control whether an element spans the full section width or aligns with the inner content.
