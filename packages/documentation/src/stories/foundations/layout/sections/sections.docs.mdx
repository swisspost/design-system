The `.section` class does not include any padding by default.
To maintain proper spacing, use a `.container` element and apply our [spacing utility classes](/?path=/docs/facaacfd-18f1-49b4-80f1-a96680730fa0--docs).

In the example bellow, the `.section` class ensures that the background-colored wrapper never exceeds 1440px.
Meanwhile, the `.container` class keeps the content aligned consistently with the rest of the page.
