# role="list"

<div className="lead">
  The `role="list"` attribute identifies an element as a list for assistive technologies, signaling
  it contains related items marked with `role="listitem"`.
</div>

<div>
While native elements like `<ul>` have this role implicitly, adding `role="list"` to non-semantic elements (e.g., `<div>`) ensures custom components are recognized as lists by screen readers.
</div>
