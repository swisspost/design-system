The listbox component displays a scrollable list of selectable options. It supports both single and multiple selection modes, making it suitable for dropdowns and multi-select menus.

The `<post-listbox>` is a container for `<post-listbox-item>` components.

<Canvas sourceState="shown" of={ListboxStories.Default} />

<Controls of={ListboxStories.Default} />

## Installation

The `<post-listbox>` element is part of the `@swisspost/design-system-components` package.
For more information, read the [getting started with components guide](/?path=/docs/edfb619b-fda1-4570-bf25-20830303d483--docs).

## Examples

### Single Select

The listbox component defaults to single selection unless the `multiselect` option is enabled. It follows
the 'Selection Follows Focus' principle, where the focused element is also the selected one. Keyboard
users can enter the component with the `tab` key and focus/select options using the `up arrow` and `down
arrow` keys.

<Canvas sourceState="shown" of={ListboxStories.PostListboxSingleSelect} />

### Multiselect

The multiselect version allows users to select multiple options. Keyboard users can enter the component using the `tab` key, navigate options
with the arrow keys, and toggle selection(s) with the `space` key.

<Canvas sourceState="shown" of={ListboxStories.PostListboxMultiselect} />

### Hidden Label

The label can be visually hidden but should always be present for accessibility reasons. Set the `label-hidden`property
to `true`to visually hide the label.

<Canvas sourceState="shown" of={ListboxStories.PostListboxNoLabel} />

### Highlighted Search Term

Passing the `searchTerm` property from a parent component (e.g., search) enables highlighting of matching
text within the listbox items.

<Canvas sourceState="shown" of={ListboxStories.PostListboxHighlightedSearch} />

### Styling

<strong>--post-listbox-label-gap</strong>

Define the gap between the label and the list items using the `--post-list-label-gap` property (
e.g., `--post-listbox-label-gap: 2rem;`).

<strong>--post-listbox-item-gap</strong>

Define the gap of the list items using the `--post-list-item-gap` property (e.g.
`--post-listbox-item-gap: 1rem 0.5rem;`).

<strong>--post-listbox-highlight-color</strong>

Specifies the color used to highlight the searched term (e.g. `--post-listbox-highlight-color: #ebebeb;`).
