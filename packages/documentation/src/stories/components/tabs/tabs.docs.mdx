## Navigation Variant

#### Indicating the Current Page

**The `aria-current="page"` attribute highlights the currently active page.**

This attribute must be manually added to the anchor element that represents the current page since it is not set automatically. The component detects which link has `aria-current="page"` and applies the active styling to the parent `post-tab-item`.

<Canvas of={TabStories.ActiveNavigationItem} />

Navigation is handled by your application's router. When users click a tab, they navigate to the URL specified in the anchor's `href` attribute.

## Panels Variant

#### Initially Active Tab

## Full-width Tabs

Setting the `full-width` property to `true` allows the tabs container to stretch across the full width of the screen. The tabs and their content remain aligned with the main page content. This property works with both variants.
