## Examples

### Initially Active Tab

The initial selection is set to the first tab by default, displaying its associated panel.
To have a different panel displayed initially,
use the `active-panel` property and assign it the name of the desired panel.

<Canvas of={TabStories.ActivePanel} />

### Full-width Tabs
