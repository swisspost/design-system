# Rating

Our Rating Web Component allows you to easily integrate 
a customizable star rating system, providing an 
interactive and visually appealing way to gather and display user feedback.

## Installation

The `<post-rating/>` element is part of the `@swisspost/design-system-components` package.
For more information, read the <LinkTo kind="getting-started-components" story="docs">getting started with components guide</LinkTo>.

## Examples

### Disabled

Is the disabled property set, the component has a clear disabled style, 
so the user can still see the current rating but is not able to interact with the component.
  

<Canvas of={RatingStories.Disabled} />
<div className="hide-col-default">
  <Controls of={RatingStories.Disabled} />
</div>

### Readonly

Is the readonly property set, is the component not interactive
but the current rating is still visible as in the default state.

<div className="hide-col-default">
