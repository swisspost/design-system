## Concrete Examples of Application

The following examples show the different characteristics of the component. These can differ in the type of visualization, the HTML structure, as well as when, how and why they are displayed.

### Floating Label

Wrap a pair of `<textarea>` and `<label>` elements in a `.form-floating` container to enable floating labels.<br/>
But note that the `<textarea>` element must come first, so we can ensure the correct styles.

<Canvas of={TextareaStories.FloatingLabel} />

### Validation

Simply add the classes `.is-valid` or `.is-invalid` to the `<textarea>` element to show it in the expected state.<br/>
When the component has been validated, don't forget to add a `<p className="valid-feedback">...</p>` or `<p className="invalid-feedback">...</p>` element after the `textarea` element to explain what went wrong. This enables the user to correct the mistake.
