## `<post-list>`

<Controls of={ListStories.Default} />

<p>
  Define the gap of the list items using the --post-list-item-gap custom property. e.g.
  `--post-list-item-gap: 1rem 0.5rem;`
</p>

<p>
  Define the gap between the title/heading and the list items using the --post-list-heading-gap
  custom property. e.g. `--post-list-heading-gap: 2rem;`
</p>

### Variants

title-hidden
horizontal
--post-list-item-gap:
--post-list-heading-gap
