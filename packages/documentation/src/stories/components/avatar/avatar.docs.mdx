## Examples

### Size

You can change the size of the avatar using the `size` property.

<Canvas of={AvatarStories.Size} />

### Picture

By default, the avatar picture displays the user's initials.
However, if you provide the `email` property for a user who owns a Gravatar,
their preferred profile picture will be used instead.

For more information, see the <a href="/?path=/docs/09aac03d-220e-4885-8fb8-1cfa01add188--docs">post-avatar-picture documentation</a>.
