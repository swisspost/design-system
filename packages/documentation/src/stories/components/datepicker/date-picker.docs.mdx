## Examples

### Inline calendar

Inline calendars are rendered directly in the page content, rather than being attached to an input. They are useful for dashboards, forms, or any context where a persistent calendar view is needed.

<Canvas sourceState="hidden" of={DatePickerStories.Inline} />

### Range input calendar

Range input calendars allow users to select a start and end date. These are ideal for booking systems, event planners, or any scenario where date ranges are relevant.

## Disabling dates

The Date Picker provides a `renderCellCallback` property, which is triggered for each calendar cell when rendering. It allows you to dynamically modify the state of individual cells, including disabling certain dates or applying custom classes. This callback essentially acts as an event that fires during cell render, giving full control over accessibility and visual presentation.

For more details, you can refer to the [AirDatepicker `onRenderCell` documentation](https://air-datepicker.com/docs?scrollTo=onRenderCell).
