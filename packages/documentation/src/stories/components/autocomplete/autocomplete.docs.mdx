<p className="lead">
  Combine a text input with a listbox to help users quickly find and select an option.
</p>

The `<post-autocomplete>` component enhances a slotted `<input>` element and works together with a
`<post-listbox>` containing `<post-listbox-option>` items.

By default, the component filters the listbox items based on the typed text. If you need to fetch
results asynchronously or apply your own filtering logic, you can listen to the cancelable
`postFilteringEvent` and prevent the default behavior.
