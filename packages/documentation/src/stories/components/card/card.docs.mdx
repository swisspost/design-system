## Interactivity

Card elements can contain many things, they can be interactive or not.
Interactive cards have a shadow and an hover effect.

To make your card interactive, simply use a `post-linkarea` component as container.
To know more about it, read our [post-linkarea documentation](/?path=/docs/1d52b794-768b-464e-90eb-4fd15774aa90--docs).

<post-banner>
  Interactive cards must include exactly one anchor element.

  Cards that require more than one link should use a non-interactive card, which allows a broader range of interactive elements.
</post-banner>

## Hidden Link

For accessibility, an interactive card must always contain a link so it can be discovered by screen readers.
The link may be visually hidden so it does not appear within the card.
