# Card

<p className="lead">
  Cards provide a flexible and extensible content container with multiple variants and options.
</p>

A card is a flexible and extensible content container.<br/>
Built with flexbox, they offer easy alignment and mix well with other components.

Cards have no fixed width to start, so theyâ€™ll naturally fill the full width of its parent element.<br/>
Use our <a href="#sizing" target="_self">sizing examples</a> to customize the size of a card.

Last but not least, our card component is based on the <a href="https://getbootstrap.com/docs/5.1/components/card/" target="_blank">bootstrap card</a> component.<br/>
Please also have a look at their documentation if you want to get more details or see even more examples.

## SCSS Imports

To import all styles from the design-system-styles package:

```css dark
@use '@swisspost/design-system-styles/index.scss';
```

To import only the styles required for this component:

```css dark
@use '@swisspost/design-system-styles/basics.scss';
@use '@swisspost/design-system-styles/components/card.scss';
```

### List Groups

Create lists of content in a card with a flush list group.<br/>
Simply replace or complement the `.card-body` with a `.list-group` element.
