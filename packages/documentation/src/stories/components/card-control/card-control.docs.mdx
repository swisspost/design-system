## Examples

### Custom icon

You can use our built-in icons by using our `post-icon` component.
If this is not sufficient, you can also use a custom icon by providing a `<svg>` or `<img>` element.
See how this can be achieved in the following example.

### Grouping

Multiple instances of the component can be grouped together using a `<fieldset>` and `<legend>` element to properly label the group.

<post-banner type="info">
  <p slot="heading">Accessible validation</p>
  <p>Group validation differs slightly from individual validation. If there is an error, you should add the following attributes to the `<fieldset>` element instead of adding them to the `<input>` elements themselves.</p>
  <ul className="unordered-list">
    <li>
      <code>aria-invalid</code>
    </li>
    <li>
      <code>aria-describedby</code>
    </li>
    <li>
      <code>disabled</code>
    </li>
  </ul>
</post-banner>

### Lined up

If you use this component multiple times in a grid and some instances contain more content than others, their heights will differ.
To align them, simply add the class `.h-full` to all of them.
