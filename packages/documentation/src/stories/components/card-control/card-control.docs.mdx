## Examples

### Custom icon

You can use our built-in icons by using our `post-icon` component.
If this is not enough, you can also use a custom icon, by implementing an `<svg>` or `<img>` element.
See how this can be acchieved in the following example.

### Grouping

The component can be grouped together. Use a `fieldset` and a `legend` element combination to label the group.

<post-banner type="info">
  <p slot="heading">Accessible validation</p>
  <p>Group validation differs a little from individual validation. If there is an error, you should add the following attributes to the `<fieldset>` element, instead of adding them to the `<input>` elements itself.</p>
  <ul className="unordered-list">
    <li>
      <code>aria-invalid</code>
    </li>
    <li>
      <code>aria-describedby</code>
    </li>
    <li>
      <code>disabled</code>
    </li>
  </ul>
</post-banner>

### Lined up

If you use this component multiple ties in a grid and some of them have more content than others, the heights will differ.
To line them up, simply add the class `.h-full` to all of them.
