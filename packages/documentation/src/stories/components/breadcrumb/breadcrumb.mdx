# Post Breadcrumb

<div className="lead">
    The breadcrumb is a secondary navigation pattern that helps users understand the hierarchy among levels and navigate back through them.
</div>

The `<post-breadcrumb>` is a container for `<post-breadcrumb-item>` components, used to display a navigational trail, showing the user's location within the app and enabling quick access to parent pages.
