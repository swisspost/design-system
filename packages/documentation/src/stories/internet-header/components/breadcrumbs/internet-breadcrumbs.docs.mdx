# Breadcrumbs

The breadcrumbs element usually needs to be placed in a container so it only spans the width of the content.

```html
<body>
  <swisspost-internet-header project="your-project-id"></swisspost-internet-header>

  <div class="container">
    <swisspost-internet-breadcrumbs></swisspost-internet-breadcrumbs>
  </div>
</body>
```
