# Alert

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.<br/>
Alerts are available for any length of text, as well as an optional close button.
By default Alerts will be rendered with a contextually appropriate icon.

## Variants

For proper styling, use one of the required contextual classes (e.g., .alert-primary, .alert-success, ...).

## Dismissible Alert

Such an alert can be removed from the UI by the user, by clicking on a dismiss button.
This button should normally be inside of the alert (as first child), but you can also use a buttom which is not included in the alerts html.
At the end, it's just a button with an `onClick` event. As this is a CSS-only component, you have to define the logic by yourself.

Dismiss buttons without text should always provide a `dismiss-label` prop, which defines the hidden text associated with the button.

## Fixed Alert

A fixed (to the bottom of the viewport) aligned alert is obtained, by adding the class `alert-fixed-bottom` to the alert element.
As this is a CSS-only component, you have to define the logic by yourself to show/hide/toggle the button.
Such an alert is normally also dismissible, so please note also the informations for a dismissible alert.

## Properties

| Name        | Property           | Example of usage                | Description                                                                                                                                                      |
| :---------- | :----------------- | :------------------------------ | :--------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Variant     | <code>class</code> | <code>alert-{variant}</code>    | Adds a style variant.<br/>Variants: <code>primary</code>, <code>succes</code>, <code>danger</code>, <code>error</code>, <code>warning</code>, <code>info</code>. |
| Custom Icon | <code>class</code> | <code>pi-{icon}</code>          | Adds a custom icon.<br/> Icons: see [Icons documentation](TODO define link)                                                                                      |
| No Icon     | <code>class</code> | <code>no-icon</code>            | Removes the icon.                                                                                                                                                |
| Dismissible | <code>class</code> | <code>alert-dismissible</code>  | Adds dismissible styles.                                                                                                                                         |
| Fixed       | <code>class</code> | <code>alert-fixed-bottom</code> | Adds fixed styles.                                                                                                                                               |
| Action      | <code>class</code> | <code>alert-action</code>       | Adds action styles.                                                                                                                                              |
